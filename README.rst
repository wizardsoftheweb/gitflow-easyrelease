``gitflow-easyrelease``
=======================

``gitflow-easyrelease`` aims to streamline ``git flow release`` commands. It adds some `semver <https://semver.org/>`__ shortcuts as well.

.. contents::

Installation
============

.. code:: sh-session

    $ pip install --user gitflow-easyrelease
